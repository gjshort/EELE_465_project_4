# Documentation

This folder is where all your documentation will go.

## Directory organization

You are free to organize your docs as you wish. I have provided the following starting point for you.

- :file_folder: [procedures](procedures): The git workflow procedure.
- :file_folder: [planning](planning): Planning documents: workload distribution, pseudocode, etc.
- :file_folder: [assets](assets): A place to put your screenshots, diagrams, etc.
